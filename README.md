# MLB 2025 Data Engineering Project

## Project Overview

This project implements a complete Extraction, Transformation and Loading (ETL) pipeline for Major League Baseball play-by-play data from the 2025 season. The solution transforms raw CSV data into a normalized PostgreSQL database schema optimized for the advanced baseball analytics.
